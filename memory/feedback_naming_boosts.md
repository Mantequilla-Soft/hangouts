Always use **Boost** / **Boosts** in all user-facing text (buttons, labels, panel titles, aria-labels, empty states, tooltips). Never use "superchat" or "Superchat" — that is a YouTube term and not the brand for this product.

**Why:** User explicitly corrected this and called it their own error to avoid. The feature was built while referring to it as a "superchat" in conversation, and that leaked into the UI text.

**How to apply:** Any time you write or review button labels, panel headers, aria-labels, placeholder text, or empty-state copy related to the boost/tipping feature — use Boost/Boosts. Applies to SDK components and any consuming app copy.
